This repository contains all orbital parameters for the constellation design results and the regional airway target information from the paper "LEO Internet Satellite Constellation Design for Regional Civil Aviation Airways With Multiple Repeat Ground Tracks."

Paper citation information:
- Han, P., Li, C., Huang, C., Huang, H., Guo, Y., & Pan, G. (2024). LEO Internet Satellite Constellation Design for Regional Civil Aviation Airways With Multiple Repeat Ground Tracks. IEEE Transactions on Aerospace and Electronic Systems.
